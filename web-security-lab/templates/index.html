🔥 Web 安全漏洞靶场

动手练习 Web 安全漏洞的发现与利用

⚠️ 仅供学习使用，请勿将此应用部署到公网！

1. SQL 注入 — 登录绕过

利用 SQL 注入绕过登录验证，无需密码即可登录

SQL Injection

2. SQL 注入 — UNION 查询

通过联合查询窃取数据库中的敏感数据

SQL Injection

3. 反射型 XSS

通过 URL 参数注入 JavaScript 代码

XSS

4. 存储型 XSS

在留言板中植入恶意脚本，持久化攻击

XSS

5. 命令注入

在 Ping 工具中注入系统命令

Command Injection

6. 文件上传漏洞

上传恶意文件（Web Shell）到服务器

File Upload

7. 路径遍历

读取服务器上的任意文件

Path Traversal

8. IDOR 越权访问

通过修改 ID 参数查看他人信息

IDOR

9. CSRF 跨站请求伪造

构造恶意请求，伪造用户操作

CSRF

10. SSRF 服务器请求伪造

让服务器发起恶意请求

SSRF

每个漏洞页面都附有提示，先自己思考，再看答案。祝你学习愉快！🚀
